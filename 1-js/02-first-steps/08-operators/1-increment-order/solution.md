- `a = 2`
- `b = 2`
- `c = 2`
- `d = 1`

```js run no-beautify
let a = 1, b = 1;

alert( ++a ); // 2, eğer toplamayı önce yaparsanız sonraki değeri gösterir.
alert( b++ ); // 1, eğer toplamayı sonra yaparsanız önceki değerini gösterir.

alert( a ); // 2, bir defa artırıldı.
alert( b ); // 2, bir defa artırıldı.
```
