# "Switch" ile yazılmış kodu "if" cümlesine çeviriniz

Aşağıdaki kodu `if..else` şekline çeviriniz.

```js
switch (tarayici) {
  case 'Edge':
    alert( "Edge browser kullanıyorsun" );
    break;

  case 'Chrome':
  case 'Firefox':
  case 'Safari':
  case 'Opera':
    alert( 'Tamam bunları destekliyoruz.' );
    break;

  default:
    alert( 'Umarım sayfanız güzel görünüyordur' );
}
```
